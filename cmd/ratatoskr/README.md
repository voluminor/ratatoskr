# ratatoskr

## Build

```bash
go build -ldflags="-s -w" -trimpath -o ratatoskr .
```

## Info & help

```bash
./ratatoskr -i
```

```bash
./ratatoskr -h
```

## Key generation

```bash
./ratatoskr -gen_private_key 1s
```

```bash
./ratatoskr -gen_private_key 10s
```

```bash
./ratatoskr -gen_private_key 1m
```
